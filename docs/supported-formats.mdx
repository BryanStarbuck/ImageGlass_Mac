# Supported Formats

ImageGlass_Mac uses macOS's native **ImageIO** pipeline as the primary
decoder, supplemented by **libvips** and **ImageMagick** for formats
ImageIO doesn't cover natively. The combined surface is well over 90
formats.

## Commonly Supported Formats

### Standard raster
- **BMP** — Bitmap image
- **GIF** — including animated GIF
- **JPEG / JPG** — including JPEG 2000 variants
- **PNG** — including APNG (animated)
- **TIFF / TIF** — including multi-frame

### Modern raster
- **AVIF** — AV1 Image File Format
- **HEIC / HEIF** — High Efficiency Image Format (non-animated only)
- **JXL** — JPEG XL
- **WebP** — including animated WebP

### Vector & document
- **SVG** — Scalable Vector Graphics (animations supported)
- **AI** — Adobe Illustrator (requires Ghostscript)
- **EPS** — Encapsulated PostScript (requires Ghostscript)
- **PDF** — Portable Document Format

### Specialized / professional
- **PSD** — Adobe Photoshop
- **ICO / CUR** — Icon and Cursor formats
- **QOI** — Quite OK Image format
- **FITS** — astronomy
- **HDR** — Radiance HDR
- **EXR** — OpenEXR

### RAW (camera)
- A wide range of RAW formats from major camera manufacturers
  (CR2/CR3, NEF, ARW, DNG, and more) via the macOS RAW pipeline.

## Read vs. Write Support

- **Read**: nearly all listed formats.
- **Save**: a focused subset — most commonly **PNG, JPEG, GIF, TIFF,
  WebP, HEIC, AVIF**.

## Optional Dependencies

Some formats require additional software to be installed alongside
ImageGlass_Mac:

| Format | Required Tool |
| --- | --- |
| AI, EPS | Ghostscript |
| HEIC / HEIF | (built-in, but **non-animated** images only) |
| APNG | (built-in) |
| Animated WebP | (built-in) |

## Special Input Methods

In addition to opening files normally, ImageGlass_Mac can read:

- **Clipboard** images and image file paths (`⌘V`).
- **Base-64 encoded text files** — useful when working with image data
  pasted from web tools or APIs.

## Adding Custom Extensions

The list of recognized extensions is **user-configurable**. Power users
can add any extension supported by the decoder pipeline by editing the
configuration files.
