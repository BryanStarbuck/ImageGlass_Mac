# Releases & News

ImageGlass_Mac is under active development. This page will list shipped
releases as they become available.

## Current Status

In development.

## Roadmap

The current development priorities follow the goals listed in
[Overview](./overview.mdx):

1. **MCP server** — automation surface for Claude Code and other MCP
   clients.
2. **Modular UI panels** — the new column with the directory / filename
   panel as the first inhabitant.
3. **Scope controls** — explicit include/exclude rules over directories,
   hierarchies, and globs.
4. **Local Storage** — plain-text, on-disk persistence of scope
   definitions and resolved file lists.
5. **MCP-driven editing of Local Storage** — Claude Code can read and
   modify scope definitions without the GUI.

See [Automation](./automation.mdx) for the architectural details.
