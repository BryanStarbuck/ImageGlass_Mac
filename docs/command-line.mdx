# Command-Line Utilities

ImageGlass_Mac ships two command-line entry points:

- **`ImageGlass_Mac`** — the main app; accepts arguments that control the
  appearance of the window at launch.
- **`igcmd`** — a separate command utility for system-level actions
  (wallpaper, default viewer registration, language/theme install, etc.).

## `ImageGlass_Mac`

Arguments are prefixed with `/` and use a `name=value` format. Any setting
exposed in `igconfig.json` can be overridden this way.

### Example

```bash
ImageGlass_Mac /ShowToolbar=false /ShowGallery=false "~/Pictures/sky.jpg"
```

This launches ImageGlass_Mac with the toolbar and gallery hidden, and
opens the file `sky.jpg`.

## `igcmd`

A dedicated utility for actions that don't require opening the viewer
window.

| Command | Description |
| --- | --- |
| `set-wallpaper <imgPath> [style]` | Set the desktop background image. `style` controls fill mode (fit, fill, stretch, tile, center). |
| `set-default-viewer [exts]` | Register ImageGlass_Mac as the default viewer for the given extensions. |
| `remove-default-viewer [exts]` | Unregister ImageGlass_Mac as the default viewer. |
| `export-frames <filePath>` | Extract every frame from an animated or multi-frame image to separate files. |
| `quick-setup` | Launch the first-run configuration wizard. |
| `check-for-update` | Check the update server for a newer release. |
| `install-languages [filePaths]` | Install one or more `.iglang` language packs. |
| `install-themes [filePaths]` | Install one or more `.igtheme` theme packs. |
| `uninstall-theme <filePath>` | Remove an installed theme pack. |

Default-viewer registration uses macOS's LaunchServices API and is
per-user.

## Common Use Cases

### Install a theme
```bash
igcmd install-themes "~/Downloads/Kobe.Jane-Smith.igtheme"
```

### Register ImageGlass_Mac as the default viewer for common types
```bash
igcmd set-default-viewer ".jpg,.jpeg,.png,.gif,.webp,.heic"
```

### Quick wallpaper change from a script
```bash
igcmd set-wallpaper "~/Pictures/Wallpapers/today.jpg" fill
```

### Open ImageGlass_Mac with a custom UI for a kiosk
```bash
ImageGlass_Mac /ShowToolbar=false /ShowGallery=false /FullScreen=true "~/Slideshow/start.jpg"
```
