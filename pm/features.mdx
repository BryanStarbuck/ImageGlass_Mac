# Features

ImageGlass_Mac is a feature-rich, simple-to-use image viewer for macOS.
The feature set spans casual viewing, animation handling, basic editing,
metadata inspection, and deep customization for power users.

## Viewing Capabilities

- Support for **90+ image formats**, including WEBP, GIF, SVG, PNG, JXL,
  HEIC, and RAW.
- **Format conversion** across up to 10 different image types.
- Multiple ways to open images:
  - Drag and drop files into the window.
  - Paste clipboard data (image files, raw bitmaps, or file paths) with
    `⌘V`.
  - Traditional file open dialog.
  - Open from base-64 encoded text files.
- **Fast thumbnail previews** of all images in the current folder
  (Gallery panel).

## Animation & Multi-Frame Support

- Animated formats — **GIF, WEBP, APNG, SVG** — are rendered natively.
- **Frame Navigation tool** for multi-frame images such as TIF and ICO:
  - Pause / resume animations.
  - Save individual frames.
  - Export all frames to separate files.

## Zoom & Display

Six zoom modes are available:

1. Auto Zoom
2. Lock Zoom
3. Scale to Width
4. Scale to Height
5. Scale to Fit
6. Scale to Fill

Additional display options:

- Adjustable **image interpolation** for upscaling / downscaling.
- **Full screen** mode.
- **Frameless** mode.
- **Window fit** mode (window resizes to image).
- **Slideshow** in a separate window with configurable countdown timers.

## Information & Metadata

- Built-in **Color Picker** tool with support for multiple color formats.
- **EXIF metadata viewer**.
- Per-image info display (dimensions, file size, format).
- **Color channel viewing** (R, G, B, A isolation).

## Basic Editing

- Rotate (90° increments).
- Flip horizontal / vertical.
- **Crop tool**.
- **Lossless compression**.
- **Image Resize** tool.

## File Management

- Set image as desktop **wallpaper** (via `igcmd`).
- Export frames from animated / multi-frame images.
- Register ImageGlass_Mac as the **default viewer** for any extension.

## Customization

- **Themes** — full visual customization including colors and icons.
- **Extension icon packs** — swap toolbar icon sets independently of theme.
- **Language packs** — UI translations.
- **Customizable layout** — toolbar contents and panel positions.
- **Custom keyboard and mouse bindings**.
- Settings driven by JSON config files (`igconfig.*.json`) — supports
  silent / bulk deployment in enterprise environments.

## Third-Party Tool Integration

- Register external editors and utilities through the Tools tab in
  Settings, or by editing `igconfig.json` directly.
- ImageGlass_Mac passes the currently displayed image's full path to the
  external tool via a `<file>` argument placeholder.
- Custom hotkeys can be bound to launch any registered tool.

## Performance

- Optimized for **minimal resource consumption** at idle and while
  browsing large folders.
- Native Apple Silicon performance with GPU-accelerated rendering via
  Metal.
