# Themes

Themes are downloadable visual packages that change the look of
ImageGlass_Mac — color accents, icon styles, panel backgrounds, dark vs.
light mode.

## What a Theme Controls

A theme bundles together:

- **Color accents** — toolbar, gallery, viewer, menus.
- **Toolbar icons** — provided as SVG so they scale crisply at any DPI.
- **App logo** within the UI.
- **Navigation arrows**.
- **Dark/light mode flag**.

## Browsing & Installing

Themes are distributed as `.igtheme` files. Two ways to install:

1. **GUI**: Open ImageGlass_Mac, go to Settings → Appearance / Themes,
   select the `.igtheme` file.
2. **Command line** — useful for scripted deployment:
   ```text
   igcmd install-themes "path/to/theme.igtheme"
   ```
   To remove:
   ```text
   igcmd uninstall-theme "path/to/theme.igtheme"
   ```

## Community Contributions

Themes are created and contributed by community members. Authors can
submit a new theme by opening an issue on the project's GitHub
repository.
